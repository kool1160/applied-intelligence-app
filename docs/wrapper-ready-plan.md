# Applied Intelligence Wrapper-Ready Plan

## Purpose

This app is being built as a strong web product now while staying easier to wrap later in a native Apple shell.

## Core direction

- iPad-first flagship experience
- iPhone as a scaled-down support experience
- web app now
- wrapper-ready later

## Current priorities

1. Strengthen the core app shell
2. Keep navigation and layout stable
3. Organize assets and metadata cleanly
4. Reduce one-off experiments in the repo
5. Preserve a clean path toward a future native wrapper

## Shell structure

The app shell centers around:
- Home
- AI-Trace
- AI-CIS
- Dashboard
- Analytics
- More / Admin

## Repo cleanup principles

- keep one primary icon source
- avoid duplicate or abandoned experiment assets
- keep metadata aligned across runtime and manifest layers
- document project direction in-repo
- make future handoff to a native shell easier, not harder

## Near-term goal

Treat this repository as the stable web-app foundation for a future Apple wrapper rather than as a temporary prototype.
