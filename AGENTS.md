# Applied Intelligence App — Project Instructions

## Architecture & Execution Constraints
- One runtime.
- One shell.
- One router.
- Native modules only.
- AI-Connect must remain a native module inside the main app.
- Do not create detached mini-app behavior.
- Do not create runtime bridges, iframes, or separate app shells.

## Scope & Change Discipline
- Keep changes tightly scoped to the requested ticket.
- Do not rewrite unrelated files.
- Preserve existing behavior unless the ticket explicitly changes it.
