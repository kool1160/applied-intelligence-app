# Theme Alignment V1

## Purpose
Align the Applied Intelligence app visual theme with the Applied Intelligence website so the app feels like part of the same premium ecosystem.

## Design Direction
- iPad optimized first
- iPhone scaled down second
- premium Apple-like operating dashboard feel
- dark navy / near-black foundation
- soft electric-blue emphasis
- cool gray-blue supporting text
- cleaner gradients
- less visual noise
- softer rounded cards and calmer glow

## Visual Goals
The app should feel:
- high-tech
- premium
- calm
- clean
- easier to scan
- visually closer to the public-facing website

## What To Change In The Theme Pass
- background theme
- card styling
- button styling
- tab styling
- text color hierarchy
- KPI card emphasis
- dashboard visual language
- analytics visual language foundation

## What Not To Change In The Theme Pass
- role logic
- status logic
- workflow rules
- metrics structure
- permissions model

## Core Theme Principle
The app should feel like a premium iPad-first operating dashboard, visually aligned with the Applied Intelligence website.

## Home / Dashboard Direction
- keep the current dashboard structure intact
- visually simplify the background
- make cards softer and more premium
- use brighter website blue for selected / primary actions
- use muted secondary actions for non-primary controls
- make KPI panels easier to scan at a glance

## Analytics Direction
Analytics should look like an advanced version of the home dashboard.
It should be:
- cleaner
- more premium
- more high-tech
- more visual
- easier to read on a shared iPad

## Priority Card Direction
Priority remains the first KPI card.
Format:
- Priority: Welding
- Reason: 6 open / 3 repeat / 2 aging

## Dashboard Structure Reminder
- Tabs: Day / Week / Month / Year
- Default: Day
- Under KPI cards: Open / Aging
- Open / Aging toggle: High Priority / Oldest First
- Open / Aging items are tappable into trace details

## Trace Detail Reminder
At the top of trace detail, show the status chain with the current step highlighted.

## Status Chain
- OPEN
- MARKED_DONE
- SUPERVISOR_VERIFIED
- CLEARED_BY_CIS
- REOPENED
