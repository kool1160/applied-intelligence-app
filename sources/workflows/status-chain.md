# Status Chain

## Day-One Status Chain
OPEN
MARKED_DONE
SUPERVISOR_VERIFIED
CLEARED_BY_CIS
REOPENED

## Meaning
- OPEN: issue has been logged and is active
- MARKED_DONE: operational user reports the issue appears fixed or complete
- SUPERVISOR_VERIFIED: supervisor checked and verified the issue
- CLEARED_BY_CIS: Continuous Improvement cleared the item from dashboard visibility
- REOPENED: issue returned or was rejected after review

## Principle
Leads report the fix.
Supervisors verify the fix.
CIS clears the dashboard layer.
